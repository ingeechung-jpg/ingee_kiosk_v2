구글 드라이브 용량 부족 알림에 미뤄뒀던 데이터 정리를 시작했다. 여기저기 흩어져 있던 수업용 공유 문서들을 정리하며, 시간이 흐르며 잊고 있던 기억들도 다시 떠오른다. 혹시 나중에 필요할까 싶은 마음에 결국 클라우드 용량을 늘리기로 했다. 용량이 넉넉해지자, 전부터 간략하게 생각만 했던 정보 모음 페이지를 만들어 보기로 결심했다. 프로그램 지식이 부족해서 생각만 하다 미뤄왔는데, AI가 하루가 다르게 발전하고 있어서 만들면서 배워야겠다 싶은 마음으로 가볍게 시작했다.

이메일, 웹사이트, 인스타그램 주소부터 수업 자료 링크들까지 이것저것 추가하다 보니, 간단한 이력도 추가하면 여러모로 편하지 않을까 생각이 들었다. 전시, 프로젝트를 덧붙이다 보니 구글 앱 스크립트로 불러오기엔 조금 버거워지나 싶었지만, 나만 볼 '개인 키오스크'라 생각하니 크게 문제 될 건 없나 싶다가도 기왕이면 무언가 기능이 있으면 좋겠어서 미팅 예약 버튼도 추가했다. 많이들 이용해 주면 좋겠는데ㅎㅎ

한쪽 화면엔 에디터를, 다른 쪽엔 웹 브라우저를 띄워 클로드 AI를 곁에 두고 낑낑대고 있을 때, 개인 웹사이트 개발을 해줬던 친구가 방문했다. 화면 이리저리 옮기며 고생하는걸 보더니 '코덱스(Codex)'라는 것을 알려준다. GPT 기반의 로컬 AI라는데, 간단히 요청만 해도 API 연결까지 알아서 처리해주어 인증 절차만 거치니 에디터를 쳐다볼 필요조차 없을 정도였다. 매번 귀찮을 법한 질문에도 기꺼이 답해주는 친구에게 고마운 마음을 전한다.

작업과 강의를 병행하며 AI가 작업 환경과 디자인 시장을 어떻게 바꿔나갈지 기대 반 걱정 반이지만, 어쨌든 새로운 환경에 적응해야 한다는 생각에 일단 시작했는데, 이전에 머릿속으로만 구상했던 작업들을 AI가 도와주니 오랜만에 주말 내내 작업만 했다. 수업 때마다 학생들에게 기록의 중요성을 강조하지만, 정작 나 자신도 기록하는 것이 어렵고 불편했는데, 이 기회에 간단한 노트 기능까지 추가하며 AI로 이것저것 만들어보는 것들을 기록해 보려고 한다.
