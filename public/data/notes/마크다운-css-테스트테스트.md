# 마크다운 샘플 문서 ㅇㅇㅇ

이 문서는 마크다운(Markdown)의 주요 기능을 보여주는 예제입니다.[^1]

[^1]: 각주 테스트

## 1. 제목 (Headings)

`#`을 사용하여 제목을 만들 수 있습니다. 1개부터 6개까지 지원합니다.

# H1 (가장 큰 제목)

## H2 (중간 제목)

### H3 (작은 제목)

## 2. 텍스트 효과 (Text Styling)

* **굵게 (Bold)**: `**텍스트**` 또는 `__텍스트__`

* *기울임 (Italic)*: `*텍스트*` 또는 `_텍스트_`

* ~~취소선 (Strikethrough)~~: `~~텍스트~~`

* `인라인 코드 (Inline Code)`: `` `코드` ``

## 3. 목록 (Lists)

### 순서 없는 목록 (Unordered List)

* 항목 1

* 항목 2

* 하위 항목 2-1

* 하위 항목 2-2

### 순서 있는 목록 (Ordered List)

1. 첫 번째 항목

2. 두 번째 항목

3. 세 번째 항목

1. ㅇㅇㅇ

2. ㄷㄷㄷ

3. ㄷㄷㄷ

## 4. 인용구 (Blockquotes)

> 이것은 인용구입니다.

> 여러 줄로 작성이 가능합니다.

## 5. 링크와 이미지 (Links & Images)

* **링크**: [Google 바로가기](https://google.com)

* **이미지**: `![대체 텍스트](이미지URL)` 형식으로 삽입합니다.
